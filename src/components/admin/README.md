# CodeQuest Admin Dashboard

## Access Instructions

To access the Admin Dashboard:
1. From the learner login page, click "Admin Login" at the bottom
2. You'll be redirected to the Admin Login page
3. Enter any credentials and click "Sign In to Admin Portal"
4. You can click "Back to Learner Login" to return to the learner interface

## Admin Features

### 1. Dashboard (Analytics)
- Real-time statistics cards showing total users, active users, courses, and revenue
- User growth line chart tracking monthly registrations
- Course enrollment bar chart by topic
- User distribution pie chart (Free vs Premium)
- Recent activity feed

### 2. User Management
- Comprehensive user table with filtering and search
- Filter by status (Active/Inactive) and role (Free/Premium)
- Actions: Send email, upgrade to Premium, activate/deactivate accounts
- User avatar, email, last active time, and join date displayed

### 3. Course Management
- Visual course cards with cover images
- Filter by topic (C/C++, Python, Java, DSA, SQL, Web)
- Toggle Featured and Hot badges
- Edit course details, pricing, and premium status
- Delete courses
- Create new courses with comprehensive form

### 4. Content Moderation
- Review forum posts and user content
- Tabs for Pending, Reported, Approved, Rejected, and All posts
- Approve or reject posts
- View report counts and user engagement metrics
- Filter and search functionality

### 5. Notifications & Communications
- Create announcements and notifications
- Schedule messages for future delivery
- Target specific user groups (All Users, Premium, Free, Admins)
- Preview notifications before sending
- Track sent, scheduled, and draft notifications

## Design System

- **Primary Color**: #2563EB (Blue)
- **Secondary Color**: #1E3A8A (Dark Blue)
- **Background**: #F8FAFC
- **Font**: Inter/Poppins
- **Border Radius**: 12-16px
- **Spacing**: 8px grid system

All components follow the same design language as the learner interface while maintaining a professional, data-centric admin aesthetic.
